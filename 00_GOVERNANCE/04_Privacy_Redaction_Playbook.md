# Privacy & Redaction Playbook

## Redact list (minimum)
- Personal identifiers: full names (unless consent), emails, phone numbers, addresses.
- Account numbers, API keys, passwords, private links, internal company info.
- Health details, intimate relationship details, minorsâ€™ info.
- Exact financial holdings / transactions if it can identify you or create risk.

## Safe share rules

Safe Share Rules
- Prefer templates + synthetic examples over real logs.
- If using real logs: remove identifiers, replace with placeholders, and blur dates if needed.
- Keep structure, delete content: readers learn the method without your private life.
