# Example: Marketing Campaign

**Domain:** Product marketing launch  
**Use case:** Launching a new B2B SaaS product feature with AI collaboration

This example shows how a small marketing team uses LC-OS governance to run a
3-month product launch campaign with an AI collaborator. The files below are
fully filled-in — not blank templates.

## What's in this folder

| File | Purpose |
|------|---------|
| `strategy-master.md` | Campaign philosophy, audience logic, brand boundaries |
| `running-document.md` | Current phase, active tasks, session notes |
| `canonical-numbers.md` | Budget, targets, performance data |
| `failure-log.md` | Mistakes made and fixed during the campaign |

## How to use this example

1. Read `strategy-master.md` first — understand the strategic logic
2. See how `running-document.md` tracks live state
3. Check `canonical-numbers.md` to see how budget/metrics are pinned
4. Read `failure-log.md` to see the kind of errors LC-OS catches

## Adapt it for your work

Replace the campaign details with your own. The structure — three authoritative
files plus a failure log — works for any marketing or content project.
